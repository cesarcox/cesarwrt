             <section id="module7" class="module-content">
                 <details>
                    <summary>Información Clave: Ciencia del Secado 1 (Psicrometría)</summary>
                    <div>
                        <h4>Introducción a la Ciencia del Secado</h4>
                        <p>Se basa en dos categorías: las condiciones de secado (calor, aire seco, flujo de aire) y la estructura que se seca (materiales, ensamblajes). La frase clave es "calor, aire seco, flujo de aire hacia el agua".</p>
                        <h4>Psicrometría - 3 Conceptos Clave</h4>
                        <p>Entender la psicrometría implica saber: 1) cuánta agua puede contener el aire (depende de la temperatura), 2) qué porcentaje de esa capacidad está ocupado (humedad relativa), y 3) cuánta agua hay realmente en el aire (ratio de humedad/GPP, punto de rocío, presión de vapor).</p>
                        <h4>El Diagrama Psicrométrico</h4>
                        <p>Es una herramienta gráfica para calcular y entender las relaciones entre temperatura, humedad relativa y volumen de humedad (ratio de humedad, punto de rocío, presión de vapor) en el aire. Conociendo dos de estas métricas, puedes encontrar la tercera.</p>
                        <h4>Evaporación y Presión de Vapor</h4>
                        <p>La evaporación es más rápida cuanto mayor es la diferencia entre la presión de vapor del agua (aumenta con la temperatura del agua) y la presión de vapor del aire (disminuye al secar el aire). Calentar materiales húmedos y secar el aire maximiza esta diferencia.</p>
                    </div>
                </details>

También hay que considerar las capas y vacíos en los ensamblajes. El objetivo es llevar las condiciones de secado (calor, aire seco, flujo de aire) hasta donde el agua pueda escapar.</p>

Clase 1 es la menor evaporación, Clase 3 la mayor, y Clase 4 implica materiales difíciles de secar.              <section id="module10" class="module-content">
                 <details>
                    <summary>Información Clave: Movimiento del Aire</summary>
                    <div>
                        <h4>Movimiento del Aire</h4>
                        <p>El flujo de aire rápido es crucial para barrer el vapor fuera de los ensamblajes y transferir energía (calor) a los materiales húmedos, acelerando la evaporación. La velocidad del aire genera presión, forzando el aire seco a entrar y el húmedo a salir.</p>
                        <h4>Configuración de Motores de Aire</h4>
                        <p>Colocar el número correcto de motores de aire, en la ubicación y ángulo adecuados es fundamental. El objetivo es lograr un flujo de aire rápido sobre todas las superficies húmedas para barrer el vapor y transferir calor. Las fórmulas del S500 ayudan a determinar la cantidad inicial necesaria.</p>
                    </div>
                </details>

                    <div>
                        <h4>Remoción Controlada de Materiales</h4>
                        <p>El objetivo es mitigar el daño removiendo la menor cantidad de material posible. No intentes secar materiales no restaurables o no descontaminables. Realiza cortes limpios y considera cómo se reemplazarán los materiales.</p>

                        <p>Limpia antes de encender los motores de aire y filtra el aire durante el secado para reducir partículas en el aire. Usa filtros HEPA y calcula la cantidad de dispositivos de filtración de aire (AFD) necesarios según los cambios de aire por hora (ACH) deseados.</p>
                        <h4>Visión General de Agua Cat 2 y 3</h4>
                        <p>Ambas son contaminadas y requieren descontaminación antes del secado. Prioriza la seguridad con PPE. Contén el área (barreras físicas y flujo de aire negativo). Remueve materiales porosos saturados (acolchado, aislamiento, paneles de yeso), limpia escombros, aplica detergente, enjuaga/extrae y aplica biocida (con consentimiento).</p>

                        <p>Se aplica específicamente a madera maciza sobre subsuelo de madera.                         <h4>Secado de Ensamblajes Aislados</h4>
                        <p>Requiere llevar aire caliente y seco al agua atrapada. Considera los componentes (madera, acero, tipo de aislante, barreras de vapor, paneles de yeso). Las estrategias van desde remover todo lo húmedo (más destructivo), ventilar con agujeros, hasta inyectar aire en la cavidad (menos destructivo).</p>
                        <h4>Secado de Sótanos ('Crawlspaces')</h4>
                        <p>El objetivo principal es secar la estructura del sótano. Extrae el agua estancada, instala una barrera de vapor sobre el suelo húmedo para detener la evaporación del suelo, y crea una cámara de secado con deshumidificadores, motores de aire y calor si es necesario.</p>

             <section id="module15" class="module-content">
                 <details>
                    <summary>Información Clave: Repaso y Fórmulas</summary>
                    <div>
                        <h4>Control de Humedad para el Secado</h4>
                        <p>Implica eliminar el vapor de agua del aire, ya sea mediante secado abierto (ventilar) o cerrado (usar deshumidificadores). El aire más seco acelera el secado y reduce daños. Los deshumidificadores LGR son los más comunes y eficientes entre 70-90°F.</p>
                        <h4>Configuración de Deshumidificadores</h4>
                        <p>Se usan para calentar y secar el aire dentro de cámaras de secado (áreas delimitadas para concentrar el secado). El cálculo inicial de deshumidificadores se basa en la Clase de Agua y el volumen del área, buscando que la tasa de deshumidificación supere la de evaporación.</p>
                        <h4>Control de Temperatura</h4>
                        <p>Es necesario gestionar tanto la temperatura del aire como la de los materiales. Se puede calentar para acelerar la evaporación (directamente o calentando el aire) y mantener el aire entre 70-90°F para la eficiencia de los deshumidificadores LGR. Usa primero el HVAC del edificio si es posible.</p>
                        </div>

        // --- Variables de Estado del Quiz ---
        let questionStates = {}; // Stores the state of each answered question in modules 1-15
        let moduleScores = {}; // Stores scores per module 1-15

        // --- Lógica de Navegación ---
        const moduleLinks = document.querySelectorAll('.module-link');
        const moduleContents = document.querySelectorAll('.module-content');
        const moduleTitle = document.getElementById('module-title');
        const initialMessage = document.getElementById('initial-message');
        const scoreDisplay = document.getElementById('quiz-score');
        const resetTotalButton = document.getElementById('reset-total-button');

        // Función para mostrar un módulo específico
        function showModule(moduleId) {
            moduleContents.forEach(content => content.classList.remove('active'));
            if (initialMessage) initialMessage.style.display = 'none';

            const selectedModule = document.getElementById(moduleId);
            const selectedLink = document.querySelector(`.module-link[data-module="${moduleId}"]`);

            if (selectedModule && selectedLink) {
                selectedModule.classList.add('active');
                moduleTitle.textContent = selectedLink.textContent;
                moduleLinks.forEach(link => {
                    link.classList.remove('bg-gray-700', 'font-semibold');
                    if(link.getAttribute('data-module') === moduleId) {
                        link.classList.add('bg-gray-700', 'font-semibold');
                    }
                });
                if (moduleId === 'module16') {
                    loadFinalTest(moduleId);
                } else {
                    loadQuiz(moduleId);
                }
            } else {
                 if(initialMessage) initialMessage.style.display = 'block';
                 moduleTitle.textContent = 'Bienvenido a la Herramienta de Estudio WRT';
                 moduleLinks.forEach(link => link.classList.remove('bg-gray-700', 'font-semibold'));
            }
            updateTotalScoreDisplay();
        }

        // Añadir listeners a los enlaces de los módulos
        moduleLinks.forEach(link => {
            link.addEventListener('click', (event) => {
                event.preventDefault();
                const moduleId = link.getAttribute('data-module');
                showModule(moduleId);
                document.querySelector('main').scrollTop = 0;
            });
        });

        // --- Lógica del Quiz (Módulos 1-15) ---
        function initializeModuleScore(moduleId) {
            if (parseInt(moduleId.replace('module','')) > 15) return;
            if (!moduleScores[moduleId] || moduleScores[moduleId].total === 0) {
                const questions = quizData[moduleId] || [];
                moduleScores[moduleId] = { correct: 0, attempted: 0, total: questions.length };
            }
             let currentCorrect = 0;
             let currentAttempted = 0;
             const questions = quizData[moduleId] || [];
             questions.forEach((q, index) => {
                 const questionId = `${moduleId}-q${index}`;
                 if (questionStates[questionId]?.answered) {
                     currentAttempted++;
                     if (questionStates[questionId].correct) {
                         currentCorrect++;
                     }
                 }
             });
             if(moduleScores[moduleId]){
                moduleScores[moduleId].correct = currentCorrect;
                moduleScores[moduleId].attempted = currentAttempted;
             }
        }

        function loadQuiz(moduleId) {
            if (parseInt(moduleId.replace('module','')) > 15) return;
            initializeModuleScore(moduleId);
            const questions = quizData[moduleId] || [];
            const quizContainer = document.querySelector(`#${moduleId} .quiz-container`);
            const moduleScoreDisplay = document.getElementById(`quiz-module-score-${moduleId}`);
            const resetButton = document.getElementById(`reset-module-quiz-${moduleId}`);
            if (!quizContainer || !moduleScoreDisplay || !resetButton) return;
            // Ensure quizContainer is part of the quiz-section div
            const quizSection = document.querySelector(`#${moduleId} .quiz-section`);
            if (!quizSection) return; // Exit if quiz section not found
            quizContainer.innerHTML = ''; // Clear only the container

            if (questions.length === 0) {
                quizContainer.innerHTML = '<p class="text-gray-500">Aún no hay preguntas de autoevaluación para este módulo.</p>';
                moduleScoreDisplay.style.display = 'none';
                resetButton.style.display = 'none';
                return;
            }

            moduleScoreDisplay.style.display = 'block';
            resetButton.style.display = 'inline-block';

            const newResetButton = resetButton.cloneNode(true);
            resetButton.parentNode.replaceChild(newResetButton, resetButton);
            newResetButton.addEventListener('click', () => resetModuleQuiz(moduleId));

            questions.forEach((q, index) => {
                const questionId = `${moduleId}-q${index}`;
                const questionState = questionStates[questionId] || { answered: false };
                const questionDiv = document.createElement('div');
                questionDiv.classList.add('quiz-question');
                questionDiv.setAttribute('data-question-id', questionId);
                const questionText = document.createElement('p');
                questionText.classList.add('mb-3', 'font-semibold');
                questionText.textContent = `Pregunta ${index + 1}: ${q.question}`;
                questionDiv.appendChild(questionText);
                const optionsDiv = document.createElement('div');
                optionsDiv.classList.add('quiz-options');
                q.options.forEach(optionText => {
                    const button = document.createElement('button');
                    button.innerHTML = `<span class="feedback-icon"></span><span>${optionText}</span>`;
                    optionsDiv.appendChild(button);
                    const iconSpan = button.querySelector('.feedback-icon');
                    if (questionState.answered) {
                        button.disabled = true;
                        if (optionText === q.correctAnswer) {
                            button.classList.add('correct');
                            if(iconSpan) iconSpan.textContent = '✔️';
                        }
                        if (optionText === questionState.selected && !questionState.correct) {
                            button.classList.add('incorrect');
                             if(iconSpan) iconSpan.textContent = '❌';
                        }
                    } else {
                         button.addEventListener('click', () => checkAnswer(moduleId, q, optionText, button, questionDiv, questionId));
                    }
                });
                questionDiv.appendChild(optionsDiv);
                const explanationDiv = document.createElement('div');
                explanationDiv.classList.add('quiz-explanation');
                explanationDiv.innerHTML = `<strong>Explicación:</strong> ${q.explanation}`;
                questionDiv.appendChild(explanationDiv);
                if (questionState.answered) {
                     explanationDiv.classList.add('visible');
                }
                quizContainer.appendChild(questionDiv);
            });
            updateModuleScoreDisplay(moduleId);
        }

        function checkAnswer(moduleId, questionData, selectedOption, buttonElement, questionContainer, questionId) {
            if (questionStates[questionId]?.answered) return;
            const optionsContainer = buttonElement.parentElement;
            const explanationDiv = questionContainer.querySelector('.quiz-explanation');
            const allButtons = optionsContainer.querySelectorAll('button');
            const isCorrect = (selectedOption === questionData.correctAnswer);
            if (!questionStates[questionId]) {
                 initializeModuleScore(moduleId);
                 moduleScores[moduleId].attempted++;
                 if (isCorrect) {
                    moduleScores[moduleId].correct++;
                 }
            }
             questionStates[questionId] = {
                answered: true,
                correct: isCorrect,
                selected: selectedOption
            };
            allButtons.forEach(btn => {
                btn.disabled = true;
                const optionText = btn.querySelector('span:last-child').textContent;
                const iconSpan = btn.querySelector('.feedback-icon');
                 if (!iconSpan) return;
                if (optionText === questionData.correctAnswer) {
                    btn.classList.add('correct');
                    iconSpan.textContent = '✔️';
                }
                if (btn === buttonElement && !isCorrect) {
                    btn.classList.add('incorrect');
                    iconSpan.textContent = '❌';
                } else if (btn !== buttonElement && optionText !== questionData.correctAnswer) {
                     iconSpan.textContent = '';
                }
            });
            if (explanationDiv) {
                explanationDiv.classList.add('visible');
            }
            updateModuleScoreDisplay(moduleId);
            updateTotalScoreDisplay();
        }

         function updateModuleScoreDisplay(moduleId) {
             if (parseInt(moduleId.replace('module','')) > 15) return;
             const scoreElement = document.getElementById(`quiz-module-score-${moduleId}`);
             if (scoreElement && moduleScores[moduleId]) {
                 const score = moduleScores[moduleId];
                 const totalQuestions = score.total > 0 ? score.total : (quizData[moduleId] || []).length;
                 scoreElement.textContent = `Puntuación del Módulo: ${score.correct} / ${totalQuestions} (Intentadas: ${score.attempted})`;
             } else if (scoreElement) {
                  const questions = quizData[moduleId] || [];
                  scoreElement.textContent = `Puntuación del Módulo: 0 / ${questions.length} (Intentadas: 0)`;
             }
         }

         function updateTotalScoreDisplay() {
             if (scoreDisplay) {
                 let totalCorrect = 0;
                 let totalAvailable = 0;
                 for (let i = 1; i <= 15; i++) {
                     const modId = `module${i}`;
                     if(quizData[modId]) {
                        totalAvailable += quizData[modId].length;
                     }
                 }
                 totalCorrect = 0;
                 for (const questionId in questionStates) {
                     const match = questionId.match(/^module(\d+)-q\d+$/);
                     if (match && parseInt(match[1]) <= 15) {
                         if (questionStates[questionId]?.correct) {
                             totalCorrect++;
                         }
                     }
                 }
                 scoreDisplay.textContent = `Puntuación Total: ${totalCorrect} / ${totalAvailable}`;
             }
         }

        // --- Final Test Logic (Module 16) ---
        function loadFinalTest(moduleId) {
            const container = document.getElementById('final-test-container');
            if (!container) return;
            container.innerHTML = '';
            finalTestData.forEach((item, index) => {
                const div = document.createElement('div');
                div.classList.add('final-test-item');
                const questionP = document.createElement('p');
                questionP.classList.add('final-test-question');
                let cleanQuestion = item.question.replace(/\?.*$/, '').trim();
                if (!cleanQuestion.startsWith('¿')) {
                    cleanQuestion = '¿' + cleanQuestion;
                }
                 if (!cleanQuestion.endsWith('?')) {
                    cleanQuestion = cleanQuestion + '?';
                }
                cleanQuestion = cleanQuestion.replace('??','?');
                questionP.textContent = `${index + 1}. ${cleanQuestion}`;
                const button = document.createElement('button');
                button.classList.add('reveal-answer-button');
                button.textContent = 'Mostrar Respuesta';
                const answerDiv = document.createElement('div');
                answerDiv.classList.add('final-test-answer');
                answerDiv.textContent = item.answer;
                button.addEventListener('click', () => {
                    answerDiv.classList.toggle('visible');
                    button.textContent = answerDiv.classList.contains('visible') ? 'Ocultar Respuesta' : 'Mostrar Respuesta';
                });
                div.appendChild(questionP);
                div.appendChild(button);
                div.appendChild(answerDiv);
                container.appendChild(div);
            });
        }

        // --- Reset Functions ---
        function resetModuleQuiz(moduleId) {
             if (parseInt(moduleId.replace('module','')) > 15) return;
            if (!quizData[moduleId]) return;
            quizData[moduleId].forEach((q, index) => {
                const questionId = `${moduleId}-q${index}`;
                delete questionStates[questionId];
            });
             if (moduleScores[moduleId]) {
                moduleScores[moduleId].correct = 0;
                moduleScores[moduleId].attempted = 0;
             } else {
                 initializeModuleScore(moduleId);
             }
            loadQuiz(moduleId);
            updateTotalScoreDisplay();
        }

        function resetTotalQuiz() {
            questionStates = {};
            moduleScores = {};
            const activeModuleLink = document.querySelector('.module-link.bg-gray-700');
            const activeModuleId = activeModuleLink ? activeModuleLink.getAttribute('data-module') : null;
            if (activeModuleId) {
                 if (parseInt(activeModuleId.replace('module','')) <= 15) {
                     loadQuiz(activeModuleId);
                 } else if (activeModuleId === 'module16') {
                     loadFinalTest(activeModuleId);
                 }
            }
            for (let i = 1; i <= 15; i++) {
                const modId = `module${i}`;
                 if (document.getElementById(`quiz-module-score-${modId}`)) {
                    initializeModuleScore(modId);
                    updateModuleScoreDisplay(modId);
                 }
            }
            updateTotalScoreDisplay();
        }

        // Add listener to the total reset button
        if(resetTotalButton) {
            resetTotalButton.addEventListener('click', resetTotalQuiz);
        }

        // Initialize the view on page load
        updateTotalScoreDisplay();
